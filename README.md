# Million Dollar Insta Page

An interactive web project inspired by the famous Million Dollar Homepage concept. This platform allows users to claim and customize individual pixel blocks, creating a unique collaborative digital advertising space. Users can preview their selected pixels on a checkout page with data managed using Local Storage for a smooth and dynamic user experience.

This project was developed during my first internship at Parkqwik Private Limited. Based on my performance and skills in previous tasks, I was assigned the responsibility of building this project, which helped me strengthen my front-end development and UI design abilities.

The project demonstrates strong front-end development skills, creative UI design, and JavaScript-based interactivity. It includes multiple sections such as Home, Contact, Blog, Dashboard, Buy Pixels, and Top Pixel Buyers, providing a complete and engaging website structure.

## Features

* Interactive pixel selection system
* Pixel customization and preview functionality
* Checkout page with Local Storage integration
* Responsive and creative user interface
* Dashboard and leaderboard-style sections
* Smooth front-end interactions
